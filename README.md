# Desafio - Classe Herói do Jogo 🧙‍♂️⚔️

Este é um pequeno projeto em JavaScript criado como parte de um desafio de programação.

## 💡 Objetivo
Criar uma classe que representa um herói de um jogo de aventura com as seguintes propriedades:

- Nome
- Idade
- Tipo (mago, guerreiro, monge, ninja)

O herói possui um método `atacar()` que imprime uma mensagem baseada no tipo de herói.

## 🛠️ Como executar

1. Abra este repositório no Codespaces ou localmente no VSCode.
2. Certifique-se de ter o Node.js instalado.
3. No terminal, execute:

```bash
node heroi.js
